# scRNAseq-hematopoiesis-analysis

This project performs single-cell RNA sequencing analysis of
hematopoietic cells using the Seurat workflow.

## Objectives
- Perform quality control and filtering
- Identify transcriptionally distinct cell populations
- Visualize cellular heterogeneity using UMAP
- Identify cluster marker genes
- Annotate hematopoietic cell types

## Methods
Analysis follows the Seurat guided clustering workflow:
https://satijalab.org/seurat/articles/pbmc3k_tutorial.html
